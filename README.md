# Flutter Internship Coding Test

This project is part of the Flutter Developer Internship coding test.

## Tech Stack
- Flutter
- Node.js + Express
- SQL Server
- JWT Authentication

## Features
- User authentication (Login, Signup)
- Category CRUD with search and debounce
- Product CRUD with pagination, search, sort, and category filter
- JWT-protected APIs

## Development
Feature branch: feature/product-crud  
Used for product CRUD and JWT protection.
